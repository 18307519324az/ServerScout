# Demo Video Script

Target length: 2-3 minutes.

Recommended format: MP4 with AI voice-over and burned-in English subtitles.

## 0:00-0:20 Opening

Subtitle:

ServerScout AI Risk Briefing Assistant is an AI-assisted security reporting workflow for small teams that need to understand scan results quickly.

This demo uses the verified local build and the packaged prototype.

## 0:20-0:55 Problem

Subtitle:

Small teams often receive technical scan data, but they still need a clear answer: what should be fixed first, why it matters, and how to explain the risk to stakeholders.

## 0:55-1:35 Product Demo

Subtitle:

ServerScout includes dashboard, asset management, scan tasks, vulnerability management, topology, attack-surface visualization, external intelligence, report export, and the new AI Briefing page.

Subtitle:

The integrated AI Briefing page can load recent ServerScout data or accept security evidence in any readable format.

Subtitle:

Click Generate AI Brief. The backend validates the input, extracts security signals, and returns input-specific sections.

Subtitle:

Change the evidence and the output changes. Unrelated input is rejected instead of producing a misleading security report.

## 1:35-2:10 AI Value

Subtitle:

The AI layer does not replace scanning. It uses a configured language model when available and clearly labels the local security analysis fallback when no model endpoint is configured.

Subtitle:

This is useful for small businesses, internal IT teams, freelance security testers, and junior security engineers.

## 2:10-2:40 Business and Roadmap

Subtitle:

The business model starts with free local use, then grows into team subscriptions, per-report pricing, and consultant packages.

Subtitle:

Next steps are generating PDF report sections directly from AI output and adding organization-specific remediation templates.

## 2:40-3:00 Closing

Subtitle:

For HackOnVibe, the focus is a practical AI micro-product: a risk briefing assistant that helps small teams move from scan output to remediation action.
