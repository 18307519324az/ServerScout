# About ServerScout

## 中文
ServerScout 是一个面向安全运营的攻击面与漏洞运营平台，重点解决从扫描到处置的协同效率问题。

核心能力包括：
- 资产发现与资产台账管理
- 扫描任务编排与实时进度跟踪
- 漏洞关联、分级与闭环处理
- 资产拓扑与攻击面可视化
- 外部情报查询与报告导出

设计目标：
- 在统一界面内，实现可观测、可追踪、可落地的扫描到修复流程。

---

## English
ServerScout is a security operations platform focused on attack-surface visibility and vulnerability workflow execution.

Core capabilities:
- asset discovery and inventory
- scan orchestration and progress tracking
- vulnerability correlation, grading, and closure workflow
- topology and attack-surface visualization
- intel lookup and report export

Design goal:
- make the scan-to-remediation workflow observable, traceable, and operationally actionable in one UI.
