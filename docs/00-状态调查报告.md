## 6. 风险评估

### 6.1 当前通过测试但需要关注

- **AiBriefingServiceTest**: 在 1/2 次重试时捕获到 JSON 解析异常（模拟 AI 返回格式异常），测试**预期了这种失败**，但生产环境中如果 AI 返回格式错误，重试机制可能不足。
- **NmapScannerImplTest**: 调用了 2 个实际扫描用例，需要本机安装 Nmap 才能完整执行。

### 6.2 安全审计要点

- JWT 密钥通过环境变量注入 ✅
- 密码使用 BCrypt 加密 ✅
- 登录 RSA 加密 ✅
- Rate Limiting 已实现 ✅
- **缺少**: 扫描目标合法性校验（允许私有 IP / CIDR 判断）
- **缺少**: Demo Mode 开关，无 Nmap/Nuclei 时可安全演示
