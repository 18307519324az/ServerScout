# 报告导出说明

## 概述

ServerScout 支持 PDF 和 Excel 两种格式的报告导出，基于扫描任务的关联数据生成多维分析报告。

## Excel 报告

### 技术实现

使用 Apache POI 生成 `.xlsx` 格式的 Excel 文件。

### 7 个 Sheet

| Sheet | 中文表头 | 内容说明 |
|-------|----------|----------|
| Sheet1 | 扫描概览 | 任务名称、扫描目标、扫描模式（Demo/Real）、开始/结束时间、耗时、状态 |
| Sheet2 | 资产列表 | 资产 IP、主机名、操作系统、开放端口数、严重漏洞数、状态、标签、发现时间 |
| Sheet3 | 开放端口 | 资产 IP、端口号、协议、服务名称、版本、Banner、状态 |
| Sheet4 | 漏洞列表 | CVE ID、严重等级、CVSS 评分、影响资产、影响软件、影响版本、状态、发现时间、修复建议 |
| Sheet5 | 扫描阶段 | 阶段名称、阶段编码、状态、开始时间、结束时间、耗时、错误信息 |
| Sheet6 | 风险评分 | 资产 IP、资产名称、资产暴露面分、漏洞严重性分、服务风险分、可利用性分、业务重要性分、修复扣减、最终风险分、风险等级、风险原因、修复建议 |
| Sheet7 | 修复建议 | 资产 IP、端口/服务、问题描述、修复建议、优先级 |

### 接口

```http
GET /api/v1/reports/excel?taskId={taskId}
```

返回 `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` 格式文件。

## PDF 报告

### 技术实现

使用 iText 7 生成 PDF 文件。中文字体通过 `PDF_FONT_PATH` 环境变量指定。

### 报告内容

1. **报告标题**：ServerScout 扫描安全报告
2. **扫描概览**：任务名称、目标范围、扫描模式、开始/结束时间、耗时
3. **扫描阶段**：9 阶段的状态、耗时、错误信息
4. **风险评分 Top 10**：按最终风险分降序排列的前 10 个资产评分
5. **高危漏洞列表**：CRITICAL/HIGH 级别的漏洞详情
6. **资产详情**：资产 IP、主机名、端口数量、漏洞数量
7. **修复建议**：按资产分组的安全加固建议
8. **安全声明**：固定文案

### PDF 中文乱码处理

- 通过 `PDF_FONT_PATH` 环境变量指定中文字体路径
- 默认回退路径：`/usr/share/fonts/truetype/noto/NotoSansCJK-Regular.ttc`
- 如果字体配置不正确，PDF 中文内容将无法正常渲染
- 建议使用 Noto Sans CJK 或 SimSun 等支持中文的字体

### 接口

```http
GET /api/v1/reports/pdf?taskId={taskId}
```

返回 `application/pdf` 格式文件。

## 报告下载接口实现

```java
@RestController
@RequestMapping("/api/v1/reports")
public class ReportController {

    @GetMapping("/pdf")
    public ResponseEntity<byte[]> downloadPdf(@RequestParam Long taskId) {
        byte[] content = reportService.generatePdf(taskId);
        return ResponseEntity.ok()
                .contentType(MediaType.APPLICATION_PDF)
                .header("Content-Disposition", "attachment; filename=report-" + taskId + ".pdf")
                .body(content);
    }

    @GetMapping("/excel")
    public ResponseEntity<byte[]> downloadExcel(@RequestParam Long taskId) {
        byte[] content = reportService.generateExcel(taskId);
        return ResponseEntity.ok()
                .contentType(MediaType.parseMediaType("application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"))
                .header("Content-Disposition", "attachment; filename=report-" + taskId + ".xlsx")
                .body(content);
    }
}
```

## 前端报告中心

前端 Reports 页面功能：

- 展示扫描任务列表（已完成的任务）
- 显示任务的 Demo Mode 标记
- 提供 PDF 和 Excel 下载按钮
- 空状态处理（无已完成任务时显示提示）
- 下载失败时显示错误提示

## Demo Mode 下的报告

Demo Mode 下扫描生成的任务数据同样支持报告导出，报告中扫描模式标记为"演示数据"。

## 安全声明

所有导出报告末尾包含以下固定声明（中英文）：

> 本报告仅用于授权资产安全检测、学习演示和内部风险分析。请勿对未授权目标进行扫描或测试。
>
> This report is for authorized security assessment, learning demonstration, and internal risk analysis only. Do not scan or test unauthorized targets.
